# Resona — Brand Identity

> Your mind, in tune.

Resona is the consumer-facing name for what was internally called NAO (Neuro-Agentic Orchestration). The Python package is still `nao` until a code rename is formally scheduled.

---

## Name

**Resona** — from *resonance*. Two ideas, one mark:
1. Brain rhythms (α, β bands) resonate at characteristic frequencies during focus.
2. Agents resonate *with* the user — they act in phase with cognitive state, not against it.

Pronounced **reh-SO-nah**. Stress on second syllable.

## Tagline

**your mind, in tune.**

Set lowercase, italic, soft. Never shouted, never capitalized.

## The Mark

Two interlocking sine waves that meet at a single luminous point — the *resonance node*. One wave is the brain (cream, organic). One is the agent (amber, deliberate). They are 180° out of phase except at the meeting point, where they phase-lock.

This is the only metaphor:
- A user **alone** is a single wave.
- A naive AI assistant is **noise on top**.
- Resona is **phase lock** — the agent matching the user.

### Construction
- Two cubic Bézier sine forms, mirrored across the horizontal axis.
- Center node: dark dot inside a luminous halo. The dark inner dot prevents the glow from looking "soft" at small sizes.
- Stroke widths scale with size; never below 2 px optical at 32 px tile.

### Files
- `resona-icon.svg` — 1024×1024 macOS-rounded app tile, indigo background.
- `resona-logo.svg` — horizontal lockup with wordmark + tagline.
- `resona-mark.svg` — transparent-background mark for menu bar / favicon / inline use.

## Color

| Role | Hex | Use |
|---|---|---|
| **Indigo Deep** | `#1B1B3A` | Primary background, body text on light, app tile |
| **Indigo Black** | `#0E0E22` | Tile gradient bottom, dark surfaces |
| **Amber Lock** | `#FFC857` | Agent wave, focus accents, the "click" |
| **Amber Glow** | `#FFE9B0` | Resonance node core |
| **Cream Wave** | `#F8E9C5` | Brain wave, soft contrast on indigo |
| **Mist** | `#FAFAF7` | Light backgrounds (web, docs) |
| **Slate** | `#5A5A7A` | Secondary text, tagline color |

Never use pure black (`#000`) — Indigo Black reads warmer next to amber.
Never use saturated red/green — keep the palette quiet so the focus state can do the talking.

## Typography

- **Wordmark / Display**: Inter, weight 500, letter-spacing 0.16em, all caps for "RESONA". On Apple platforms, fall through to SF Pro.
- **Body**: Inter / SF Pro, weight 400.
- **Tagline**: Inter italic, weight 300.
- **Numerals (focus scalar, latency)**: tabular figures, monospace stack (`SF Mono`, `JetBrains Mono`).

## Voice

- Calm. The product is *about* being calm — the copy must model it.
- Empirical, not mystical. We say "frontal beta over alpha", not "your inner light."
- Privacy is a refrain, not a footnote: "*Raw signal stays on your Mac.*" appears wherever we describe the data flow.
- Avoid productivity-bro language. Resona is not "10x your output." It is "let your tools meet you where you are."

## Don't

- Don't use the wordmark without space equal to the height of the "R".
- Don't tilt the mark.
- Don't recolor the waves to brand-of-the-month palettes — the cream/amber pair *is* the brand.
- Don't pair with stock "neuron firing" imagery. The mark is the only neural metaphor we need.
- Don't render the inner node as a star, sparkle, or emoji-adjacent shape. It's a dot. The dot is the whole point.
